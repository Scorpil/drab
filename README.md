# drab

## An unstyled Svelte component library

[drab.robino.dev](https://drab.robino.dev)

### MIT License

### Install

```bash
npm create svelte@latest
```

```bash
npm i -D drab
```

### Use

```svelte
<script>
	import { ShareButton } from "drab";
</script>

<ShareButton
	text="Check out this page: "
	title="drab"
	url="https://drab.robino.dev"
/>
```
